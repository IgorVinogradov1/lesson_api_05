# Сравниваем вакансии программистов

Скрипт собирает статистику по вакансиям программистов в Москве с **HeadHunter** и **SuperJob**.  
Для каждого языка программирования считает: количество найденных вакансий, обработанных с зарплатой и среднюю зарплату (RUB).

### Как установить:

1. **Клонируйте репозиторий:**

2. **Создайте виртуальное окружение:**

``
python -m venv venv
``

3. **Установите зависимости:**

``
pip install -r requirements.txt
``

4. **Создайте файл .env:**

``
SJ_TOKEN=ваш_токен_superjob
``

Получите токен на [api.superjob.ru](https://api.superjob.ru/).

5. **Запустите:**

``
python main.py
``

Результат: две таблицы с данными по языкам программирования (Python, JavaScript, Java, PHP и др.) для Москвы.
